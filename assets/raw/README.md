# Asset Intake Folder

Drop your files here before I process them.

## Where to put things

| Folder | What goes here |
|--------|---------------|
| `logo/` | Logo files in any format — SVG, PNG, AI, PDF |
| `team/` | Team and leadership photos |
| `products/fet/` | FET product photos and any diagrams |
| `products/seal/` | SEAL product photos |
| `products/coffee/` | Coffee product and brand photos |
| `products/logistics/` | Logistics / fleet / operations photos |
| `other/` | Anything else — office photos, brand materials, documents |

## File naming (don't worry too much — just drop them in)

I will rename, resize, and optimise everything after you drop them in.
Just make sure I can tell what each file is from the filename or folder.
